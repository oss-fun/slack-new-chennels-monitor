# slack-new-chennels-monitor

# What is this?

* Notify you that a new channel in slack has been created.

# Install

```
npm install
```

# Run

```
node src/app.js
```

# Settings
```
export WEB_SLACK_TOKEN="{Slack token here}"
export BOT_NAME="channel-monitor"
export TARGET_SLACK_CHANNEL="#new_channels"
```
